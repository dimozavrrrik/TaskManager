Для PWA необходимо создать следующие иконки:

- favicon.ico (16x16, 32x32)
- icon-192.png (192x192)
- icon-512.png (512x512)

Вы можете использовать онлайн генераторы иконок или инструменты дизайна для создания этих файлов.

Рекомендуемый стиль: фиолетовый (#594AE2) с белой буквой T или иконкой задачи.
